MiniLM: Deep Self-Attention Distillation for Task-Agnostic Compression of Pre-Trained Transformers1. AbstractLarge pre-trained language models (LMs) like BERT and its variants have achieved great success in NLP but suffer from high computational overhead. This paper presents MiniLM, a simple and effective approach to compress large Transformer-based pre-trained models using deep self-attention distillation. The small model (student) is trained by deeply mimicking the self-attention module of the last Transformer layer of the large model (teacher). In addition to utilizing attention distributions (the scaled dot-product of queries and keys), this work introduces the scaled dot-product between values in the self-attention module as a new form of deep self-attention knowledge. Furthermore, utilizing a Teacher Assistant helps boost distillation.Experimental results demonstrate that MiniLM retains more than 99% accuracy on SQuAD 2.0 and several GLUE benchmark tasks while using only 50% of the Transformer parameters and computation of the teacher model. It also obtains highly competitive results when applied to multilingual pre-trained models.2. Introduction & BackgroundThe Problem: State-of-the-art pre-trained language models contain hundreds of millions of parameters, making them difficult to deploy in resource-constrained or latency-sensitive environments.Task-Agnostic vs. Task-Specific Distillation: Instead of distilling a model specifically for a single downstream task, task-agnostic distillation compresses the pre-trained language model directly. This allows the compact student model to be fine-tuned on any downstream task later.Limitations of Prior Work: Previous intermediate-layer distillation techniques (such as TinyBERT or MobileBERT) rely heavily on strict, explicit layer-to-layer mapping and parameter restrictions (e.g., forcing the student to have matching hidden dimension sizes or identical numbers of attention heads).3. Core MethodologyA. Last-Layer Distillation FlexibilityRather than forcing layer-by-layer matching, MiniLM primarily distills the self-attention module from the last Transformer layer of the teacher model. This design choice completely alleviates the mapping difficulties between uneven teacher/student layer counts and allows arbitrary flexibility for the student's layer configuration.B. Two Pillars of Deep Self-Attention KnowledgeMiniLM guides the student model using two main relational matrices derived from the teacher's self-attention module:Self-Attention Distribution Transfer:The student matches the teacher's standard self-attention distributions ($A$), which represent how tokens attend to each other. This is computed using the scaled dot-product of the queries ($Q$) and keys ($K$):$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$The distillation loss minimizes the Kullback-Leibler (KL) divergence between the teacher and student attention distributions.Self-Attention Value-Relation Transfer (New Knowledge):The authors introduce a new relationship vector based on the values ($V$). They calculate the scaled dot-product between the value vectors across all pairs of tokens. Because it captures how deeply the representations relate to each other independent of the attention map, it forces a deeper behavioral mimicry.Advantage: Using value-relation matrices transforms mismatched hidden dimensions (between teacher and student) into equivalent token-to-token relation matrices, completely bypassing the need for an additional parameter projection matrix to realign student hidden states.C. Teacher Assistant (TA) DistillationFor massive pre-trained teacher models, directly distilling down to a small student can cause a significant performance drop due to capacity mismatch. MiniLM successfully utilizes a Teacher Assistant (an intermediate-sized model) to bridge the size gap, distilling from Teacher $\rightarrow$ TA $\rightarrow$ Student.4. Key Experimental ResultsBERT-Base Compression: A 6-layer student model with 768 hidden dimensions distilled using MiniLM from a $\text{BERT}_{\text{BASE}}$ teacher runs 2.0× faster while retaining 99%+ accuracy on SQuAD 2.0 and GLUE benchmarks.Multilingual Efficiency: When applied to multilingual compression using $\text{XLM-R}_{\text{BASE}}$ as the teacher, MiniLM models achieved exceptional multi-language cross-lingual transfer capabilities with a fraction of the parameter size.ReferencesWang, W., Wei, F., Dong, L., Bao, H., Yang, N., & Zhou, M.
